# 🎮 Simon Says - Color Sequence Game

A classic memory challenge game built with Python and Pygame.

## 🎯 Game Description

Simon Says is a memory game where players must remember and repeat an increasingly long sequence of colored buttons. Each round, the game adds one more color to the sequence, challenging the player's memory and attention.

## ✨ Features

- 🎨 Four colored buttons: Red, Green, Blue, Yellow
- 🔊 Visual feedback with button highlights
- 📊 Score tracking and high score persistence
- 🎯 Progressive difficulty with increasing sequence length
- 🎮 Clean, modern interface with smooth animations
- 💾 High score saved locally
